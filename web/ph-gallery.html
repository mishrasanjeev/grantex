Slide 2 / 6 — The Problem
The Status Quo
AI Agents Have No Authorization Standard

No Scoped Permissions

No standard way to give agents scoped, time-limited permissions for specific actions

Unverifiable Authorization

Services can't cryptographically verify whether an agent is actually authorized

No Audit Trail

No auditable, immutable record of what agents do on behalf of users

No Delegation Chain

No delegation model for multi-agent orchestration and sub-agent pipelines

This is where the internet was before OAuth 2.0.
Slide 3 / 6 — How It Works
5-Step Flow
How It Works
1

Register Agent & Declare Scopes

Define what your agent can do: read emails, manage calendar, execute trades

2

User Approves via Consent UI

Human-in-the-loop: the user reviews and approves requested scopes

3

Grantex Issues Signed Grant Token

RS256-signed JWT with scopes, expiry, principal ID, and audit claims

4

Any Service Verifies Offline via JWKS

Zero network calls needed — verify tokens with the public key set

5

Immutable Audit Trail + Instant Revocation

Every action is logged. Revoke any grant instantly, across all services

Slide 4 / 6 — Code Quickstart
Quick Start
5 Lines to Authorize Your Agent
TypeScript 
import { Grantex } from '@grantex/sdk';

const gx = new Grantex({ apiKey: 'gx_...' });

// 1. Request authorization
const auth = await gx.authorize({
  agentId:  'agent_123',
  userId:   'user_456',
  scopes:   ['email:read', 'calendar:write'],
});

// 2. Exchange code for grant token
const { grantToken } = await gx.tokens.exchange({
  code:    auth.code,
  agentId: 'agent_123',
});

// 3. Verify anywhere — zero network calls
const grant = await gx.tokens.verify(grantToken);
Python 
from grantex import Grantex

gx = Grantex(api_key="gx_...")

# 1. Request authorization
auth = gx.authorize(
    agent_id="agent_123",
    user_id="user_456",
    scopes=["email:read", "calendar:write"],
)

# 2. Exchange code for grant token
result = gx.tokens.exchange(
    code=auth.code,
    agent_id="agent_123",
)

# 3. Verify anywhere — zero network calls
grant = gx.tokens.verify(result.grant_token)
